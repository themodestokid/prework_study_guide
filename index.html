Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Each element is closed by a tag with the same label as the opening tag, With a slash before it. It the element has no content, you can close the element by putting a slash at the end of the opening tag. Browsers generally allow content-free elements such as <br> to be unclosed.

CSS

  • margin is space outside border, padding is space between border and contents.
  • Visual Studio Code lets you select colors visually.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Add "\[\033[01;93m\]$(__git_ps1)" to $PS1 in .bashrc to incorporate branch name into prompt.
